cli es una herramienta de línea de comandos que se utiliza para crear proyectos, agregar archivos y realizar tareas
de desarrollo en Angular.

Cuando se compila TypeScript, se genera un archivo JavaScript que es el que se ejecuta en el navegador.
Compilar es necesario para que el navegador pueda interpretar el código TypeScript.


app es un componente que consta de diferentes elementos:

-Routing (rutas) que se encuentran en el archivo app-routing.module.ts, que se encarga de redireccionar a los diferentes
componentes de la aplicación.
-Hoja de estilo CSS
-HTML
-spec.ts para realizar pruebas unitarias (testeos)
-Un archivo TypeScript que contiene la lógica del componente
-Modulos de Angular que se importan en el archivo app.module.ts
